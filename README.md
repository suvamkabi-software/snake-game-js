A classic **Snake Game** built entirely with **pure HTML, CSS, and JavaScript** — no frameworks, no dependencies, just the browser. Control the snake, eat the food, grow longer, and survive as long as you can without hitting the walls or yourself!

---

## 🎮 Live Demo

> Open `index.html` directly in your browser — no server needed!

---

## ✨ Features

- 🕹️ **Smooth Gameplay** — responsive snake movement on a grid board
- 👑 **High Score Tracking** — your best score is remembered across sessions
- 🏆 **Live Score Counter** — score updates every time the snake eats food
- ⏱️ **Timer** — tracks how long you've survived in `MM:SS` format
- 🎬 **Start Screen** — animated start modal with a play button
- 💀 **Game Over Screen** — clean game-over overlay with a restart button
- 🎨 **Stylish UI** — built with the Poppins font and polished CSS animations
- 📱 **No Installation** — runs entirely in the browser, zero setup

## 🚀 Getting Started

### Option 1 — Clone & Open

```bash
git clone https://github.com/suvamkabi-software/snake-game-js.git
cd snake-game-js
```

Then simply open `index.html` in any modern browser.

### Option 2 — Download ZIP

1. Click the green **Code** button on GitHub
2. Select **Download ZIP**
3. Extract the folder
4. Open `index.html` in your browser

## 🕹️ How to Play

| Key | Action |
|-----|--------|
| `Arrow Up` /    | Move Up |
| `Arrow Down` /  | Move Down |
| `Arrow Left` /   Move Left |
| `Arrow Right` /  | Move Right |

- Click **START GAME** on the welcome screen to begin
- Guide the snake to eat the food (🐭)
- Each food eaten increases your score and grows the snake
- Avoid hitting the walls or the snake's own body
- Click **RESTART GAME** after a game over to try again
